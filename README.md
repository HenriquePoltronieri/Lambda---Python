# 📘 Estudo de Funções Lambda em Python

Este projeto demonstra o uso de **funções lambda** em Python, além da aplicação de funções built-in como `map()`, `filter()` e `sorted()`.

---

## 🚀 Conteúdo abordado

- Funções tradicionais vs funções lambda
- Uso de `map()` para aplicar funções em listas
- Uso de `filter()` para filtrar dados
- Ordenação com `sorted()` e `lambda`
- Manipulação de listas e dicionários
- Exercícios práticos com lambda

---

## 🧠 Exemplos

def dobrar(x):
    return x * 2
